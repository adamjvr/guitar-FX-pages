# Caline

## Scope
This section covers **Caline analog or analog-style clone-derived pedals**:
- overdrive
- distortion
- fuzz
- boost
- compression
- EQ / other analog utility where the clone target is commonly reported

It **does not try to list Caline’s clearly digital time/modulation pedals here**, except where a dual/combo pedal includes them.
